### 🌟 درباره V2Ray Collector
🚀 **V2Ray Collector** یک ابزار PHP فوق‌العاده است که کانفیگ‌های V2Ray را از کانال‌های مختلف جمع‌آوری کرده و در فایل‌های اشتراکی ذخیره می‌کند. این پروژه صرفاً برای اهداف آموزشی طراحی شده و استفاده تجاری یا شخصی از آن ممنوع است!

### 🎮 ویژگی‌های کلیدی
- 🔄 جمع‌آوری خودکار کانفیگ‌ها از منابع مختلف.
- 🌐 پشتیبانی از پروتکل‌ها: SHADOWSOCKS، VLESS، VMESS، TROJAN، TUIC و HYSTERIA2.
- 🛠️ سازگاری با کلاینت‌های Xray، Singbox، Clash و Surfboard.
- 📂 فایل‌های اشتراکی منظم و دسته‌بندی‌شده.
- 🆓 متن‌باز تحت مجوز AGPL-3.0.

### 📖 راهنمای استفاده
1. لینک اشتراک را از جدول‌های زیر کپی کنید.
2. لینک را در کلاینت خود (مثل Xray یا Clash) وارد کنید.
3. مطمئن شوید کلاینت شما از پروتکل‌های مورد نظر پشتیبانی می‌کند.

## 🇬🇧 English

### 🌟 About V2Ray Collector
🚀 **V2Ray Collector** is a powerful PHP tool that gathers V2Ray configurations from various channels and organizes them into subscription files. This project is for educational purposes only—no commercial or personal use allowed!

### 🎮 Key Features
- 🔄 Automatic collection of configs from multiple sources.
- 🌐 Supports protocols: SHADOWSOCKS, VLESS, VMESS, TROJAN, TUIC, and HYSTERIA2.
- 🛠️ Compatible with clients like Xray, Singbox, Clash, and Surfboard.
- 📂 Organized subscription files by protocol and client.
- 🆓 Open-source under the AGPL-3.0 license.

### 📖 How to Use
1. Copy the subscription link from the tables below.
2. Import the link into your client (e.g., Xray or Clash).
3. Ensure your client supports the desired protocols.
4. Let’s go! 🚀
